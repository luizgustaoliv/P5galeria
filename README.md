## Descrição da reprodução

A pintura foi recriada digitalmente com **p5.js** utilizando retângulos coloridos posicionados manualmente para representar a estrutura original da obra.

Foram utilizados:

- Tons variados de vermelho, vinho e rosa
- Fundo claro semelhante ao original
- Sobreposição de formas geométricas
- Ajuste proporcional para canvas `400x400`

## Interação adicionada

Ao passar o mouse sobre qualquer faixa da pintura:

- apenas o retângulo selecionado gira aleatoriamente entre `-15°` e `15°`
- o novo ângulo permanece após sair com o mouse
- cada interação modifica a composição visual

Essa proposta mantém o estilo abstrato da obra e adiciona dinamismo sem perder a identidade original.
